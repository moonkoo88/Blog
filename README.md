
게시판 샘플 프로젝트
=======

스프링 부트를 이용한 게시판 프로젝트

환경
=================

   * JAVA8
   * SPRINGBOOT 2.0.0.M7
   * GRADLE
   * SPRING DATA JPA
   * SPRING SECURITY
   * SPRING SESSION DATA REDIS
   * LOMBOK
   * THYMELEAF
   * JOOQ
   * QUERYDSL
   * LUCY-XSS-SERVLET
